## Instructor Objectives

### 1. Understanding the Concept of Decision Structures
- Explain the importance of decision-making in programming.
- Describe how decision structures control the flow of a program.

### 2. Basic Decision-Making Statements in C
- Introduce the primary decision structures: `if`, `if-else`, and `switch`.
- Explain the purpose of each statement in program control.

### 3. Implementation of if and if-else Statements
- Demonstrate how to use `if` statements to execute code based on a condition.
- Teach the use of `if-else` statements to handle true and false conditions.

### 4. Using switch Statements for Multiple Choices
- Explain the structure of a `switch` statement.
- Guide students in applying `switch` for multiple-case scenarios.
- Compare `switch` with multiple `if-else` statements for better readability.

### 5. Understanding Relational and Logical Operators
- Teach the use of relational operators (`==`, `!=`, `<`, `>`, `<=`, `>=`).
- Explain logical operators (`&&`, `||`, `!`).
- Demonstrate how to create complex conditions using these operators.
